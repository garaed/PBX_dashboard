# PBX Dashboard

Веб-дашборд для мониторинга FreePBX / Asterisk. Устанавливается на тот же сервер что и FreePBX, доступен по порту 8080.

## Возможности

- **Транки** — статус регистрации PJSIP, активные каналы, группировка по статусу
- **Операторы** — подключённые контакты сгруппированные по endpoint, живой счётчик, поиск по коду/IP, названия групп с сохранением
- **Fail2Ban** — просмотр заблокированных IP, разбан одним кликом, массовое добавление в whitelist (DEFAULT и конкретный jail)
- **Качество** — процент дозвона с трендом, 100%-стековый график, тепловая карта по часам, рейтинг транков (находит выгоревшие номера)
- **Записи** — поиск по номеру и дате, скачивание одной записи или всех архивом

## Быстрая установка

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/garaed/pbx-dashboard/main/setup.sh)
```

Скрипт сам установит зависимости, скопирует файлы и запустит systemd-сервис.

## Требования

- Debian 12 / Ubuntu 22+ (другие дистрибутивы тоже работают)
- Python 3.10+
- FreePBX с PJSIP-транками
- Root-доступ

## После установки

Дашборд доступен по адресу: `http://IP-СЕРВЕРА:8080`

Учётные данные MySQL читаются автоматически из `/etc/freepbx.conf`.

## Управление сервисом

```bash
systemctl status  pbx-dashboard   # статус
systemctl restart pbx-dashboard   # перезапуск
journalctl -u pbx-dashboard -f    # логи в реальном времени
```

## Обновление

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/ТВОЙ_НИК/pbx-dashboard/main/setup.sh)
```

Запуск установки поверх существующей — файлы обновятся, сервис перезапустится.
